# Python 数据结构基础 — 学习笔记

## 一、变量与数据类型

变量是存储数据的容器。Python是动态类型语言，变量不需要声明类型。

基本数据类型：
- **整数（int）**：如 1, 100, -5
- **浮点数（float）**：如 3.14, -0.5
- **字符串（str）**：如 "hello", 'world'
- **布尔值（bool）**：True 或 False

类型转换使用 int(), float(), str(), bool() 等内置函数。

## 二、列表（List）

列表是Python中最常用的数据结构之一，用于存储有序的元素集合。

特点：
- 有序、可变、允许重复元素
- 用方括号 [] 定义：nums = [1, 2, 3, 4, 5]
- 支持索引访问：nums[0] 获取第一个元素
- 支持切片操作：nums[1:3] 获取索引1到2的元素
- 常用方法：append(), insert(), remove(), pop(), sort(), reverse()

时间复杂度：
- 索引访问：O(1)
- 尾部追加 append()：O(1) 均摊
- 任意位置插入/删除：O(n)
- 查找元素：O(n)

## 三、元组（Tuple）

元组与列表类似，但是不可变。

特点：
- 用圆括号 () 定义：point = (3, 4)
- 一旦创建不能修改（不可变）
- 因为不可变，所以可以作为字典的键
- 内存占用比列表小
- 适合存储不应被修改的数据（如坐标、配置常量）

## 四、字典（Dictionary）

字典是键值对（key-value）的集合，基于哈希表实现。

特点：
- 用花括号 {} 定义：student = {"name": "张三", "age": 20}
- 键必须是不可变类型（字符串、数字、元组）
- 键唯一，值可以重复
- 查找、插入、删除的平均时间复杂度都是 O(1)
- 常用方法：get(), keys(), values(), items(), update()

哈希表原理：
字典内部使用哈希函数将键映射到数组索引。当两个键产生相同的哈希值时发生哈希冲突，
Python使用开放寻址法解决冲突。

## 五、集合（Set）

集合是无序、不重复元素的集合，基于哈希表实现。

特点：
- 用花括号或 set() 定义：s = {1, 2, 3} 或 s = set([1, 2, 3])
- 元素必须可哈希（不可变类型）
- 自动去重
- 支持集合运算：交集(&)、并集(|)、差集(-)、对称差集(^)
- 成员检查 .in. 的时间复杂度为 O(1)

## 六、字符串操作

字符串是不可变的字符序列。

常用操作：
- 拼接：+ 运算符或 join() 方法
- 切片：同列表切片
- 查找：find(), index(), count()
- 分割与合并：split(), join()
- 大小写：upper(), lower(), title()
- 去除空白：strip(), lstrip(), rstrip()
- 格式化：f-string（推荐）、format()、%格式化

注意：字符串是不可变的，每次"修改"操作实际是创建新字符串。
大量拼接时应使用 join() 而不是 + 运算符（性能差异显著）。
